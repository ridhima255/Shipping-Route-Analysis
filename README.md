#  Shipping Route Analysis Dashboard

An interactive data analytics dashboard built using Streamlit to analyze shipping performance and identify delays, bottlenecks, and efficiency trends.

#Features
- KPI metrics (Avg Lead Time, Delay %, Orders)
- US Shipping Heatmap (Plotly)
- Route Efficiency Analysis
- Delay Distribution & Trends
- Filters (State, Ship Mode, Date)
- Smart Insights & Bottleneck Detection
- Interactive Data Exploration Panel

# Tech Stack
- Python
- Pandas
- Streamlit
- Seaborn & Matplotlib
- Plotly

# Key Insights
- Identifies fastest and slowest shipping routes
- Highlights delay-heavy regions
- Helps optimize logistics decisions
